NodeAuth
========
This is an example of an application creating with Node.js, this application uses following modules:

|Module            |
|------------------|
|mongodb           |
|mongoose          |
|connect-flash     |
|express-validator |
|express-session   |
|express-messages  |
|passport          |
|passport-local    |
|passport-http     |
|multer            |

It uses bootstrap as well.

Usage
-----
1. Download bootstrap and copy it to the public folder
2. `run node bin/www`
